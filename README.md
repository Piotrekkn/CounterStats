## Build

### Linux
To run this project, run the following commands:

```bash
dotnet run
```

### Windows
Make sure to setup gtk and dotnet sdk first:

[Gtk Install](https://www.gtk.org/docs/installations/windows/)

[Dotnet SDK](https://dotnet.microsoft.com/en-us/download)

To run this project, run the following commands:

```bash
dotnet run
```
